# Game FFI Skills: Mastering the Unified Annotation System

## 🎯 Quick Reference

This document provides practical examples and patterns for the unified `game_ffi` annotation system. Each pattern is battle-tested and ready to copy-paste.

**Feature Flags:**
- `unity` - Generate C# bindings
- `unreal` - Generate C++ bindings
- Both can be enabled simultaneously

---

## 📦 Core Components

### 1. Basic Component Definition

The simplest possible component:

```rust
use game_ffi::GameComponent;

#[derive(GameComponent)]
#[uuid = "11111111-2222-3333-4444-555555555555"]
pub struct Health {
    pub current: f32,
    pub maximum: f32,
}
```

**Auto-generates:**
- C# `struct Health` with correct layout
- C++ `struct FHealth` with USTRUCT macros
- FFI functions: `set_health()`, `get_health()`
- UUID for type identification

## 🚀 System Patterns

### 1. Basic System

```rust
use game_ffi::GameSystem;
use bevy_ecs::prelude::*;

#[derive(GameSystem)]
#[system(stage = Update)]
fn health_regen_system(
    mut query: Query<&mut Health>,
    time: Res<Time>,
) {
    for mut health in query.iter_mut() {
        if health.current < health.maximum {
            health.current += 10.0 * time.delta_seconds();
            health.current = health.current.min(health.maximum);
        }
    }
}
```

### 2. Changed Query (Optimization)

```rust
#[derive(GameSystem)]

fn sync_position_system(

    mut writer: FfiWriter,

    for (entity, pos) in query.iter() {
        // Only sync if position actually changed

}
```

### 3. Multiple Components Query

```rust
#[derive(GameSystem)]
#[system(stage = Update)]
fn player_movement_system(
    mut query: Query<(
        &mut PlayerPosition,
        &Rotation,
        &mut Velocity,
    )>,
    input: Res<Input>,
    time: Res<Time>,
) {
    for (mut pos, rot, mut vel) in query.iter_mut() {
        // Apply input to velocity
        vel.x = input.get_axis("MoveX") * 500.0;
        vel.z = input.get_axis("MoveZ") * 500.0;
        
        // Apply velocity to position
        let move_dir = rot.to_quat() * Vec3::new(vel.x, 0.0, vel.z);
        pos.x += move_dir.x * time.delta_seconds();
        pos.y += move_dir.y * time.delta_seconds();
        pos.z += move_dir.z * time.delta_seconds();
    }
}
```

### 4. Event-Based System

```rust
use game_ffi::GameEvent;
use bevy_ecs::event::{Event, EventReader, EventWriter};

#[derive(GameEvent)]
pub struct DamageEvent {
    pub target: Entity,
    pub amount: f32,
    pub source: Entity,
}

#[derive(GameSystem)]
#[system(stage = Update)]
fn damage_system(
    mut events: EventReader<DamageEvent>,
    mut health_query: Query<&mut Health>,
) {
    for event in events.iter() {
        if let Ok(mut health) = health_query.get_mut(event.target) {
            health.current -= event.amount;
            health.current = health.current.max(0.0);
        }
    }
}
```

### 2. Runtime Field Access

```rust
use game_ffi::ReflectRegistry;

fn get_field_by_name<T: GameComponent>(
    registry: &ReflectRegistry,
    component: &T,
    field_name: &str,
) -> Option<ReflectValue> {
    let metadata = T::metadata();
    
    for (idx, field) in metadata.fields.iter().enumerate() {
        if field.name == field_name && !field.skip {
            return Some(registry.get_field_value(component, idx as u32));
        }
    }
    
    None
}

// Usage:
if let Some(ReflectValue::Float(health)) = 
    get_field_by_name(&registry, &health_component, "health")
{
    println!("Health: {}", health);
}
```

### 3. Validation at Runtime

```rust
fn validate_component<T: GameComponent>(component: &T) -> bool {
    let metadata = T::metadata();
    
    for field in &metadata.fields {
        if field.skip {
            continue;
        }
        
        let value = unsafe {
            std::ptr::read_volatile(
                (component as *const T as *const u8).add(field.offset)
                    as *const f32
            )
        };
        
        if let Some(min) = field.min_float {
            if value < min {
                return false;
            }
        }
        
        if let Some(max) = field.max_float {
            if value > max {
                return false;
            }
        }
    }
    
    true
}
```

## 🔧 Zero-Copy FFI Patterns

### 1. Direct Memory Access

```rust
use game_ffi::FfiWriter;

fn sync_components_to_engine(
    query: Query<(Entity, &PlayerPosition)>,
    mut writer: FfiWriter,
) {
    for (entity, pos) in query.iter() {
        // Zero-copy: Direct memory access
        unsafe {
            writer.write_raw(
                entity.id(),
                pos as *const PlayerPosition as *const u8,
                std::mem::size_of::<PlayerPosition>(),
            );
        }
    }
}
```

---

### 2. Batch Writing

```rust
fn batch_sync_system(
    query: Query<(Entity, &PlayerPosition, &Rotation)>,
    mut writer: FfiWriter,
) {
    let mut batch = writer.start_batch();
    
    for (entity, pos, rot) in query.iter() {
        batch.write_component(entity.id(), pos);
        batch.write_component(entity.id(), rot);
    }
    
    batch.commit();  // Single FFI call for all components
}
```

---

### 3. Direct Component Read

```rust
use game_ffi::FfiReader;

fn read_engine_components(
    entities: &[Entity],
    mut reader: FfiReader,
    mut commands: Commands,
) {
    for entity in entities {
        if let Some(pos) = reader.read_component::<PlayerPosition>(entity.id()) {
            commands.entity(entity).insert(*pos);
        }
    }
}
```

## 🐛 Common Pitfalls & Solutions

### 1. Incorrect Padding

```rust
// BAD: No explicit padding
#[derive(GameComponent)]
pub struct BadPadding {
    pub a: u8,
    pub b: u32,  // Wrong layout!
}

// GOOD: Explicit padding
#[derive(GameComponent)]
pub struct GoodPadding {
    pub a: u8,
    pub _pad: [u8; 3],  // Explicit padding
    pub b: u32,
}
```

---

### 2. Enum Size Mismatch

```rust
// BAD: Default enum size is platform-dependent
#[derive(GameComponent)]
pub enum BadEnum {
    A = 0,
    B = 1,
}

// GOOD: Explicit size
#[derive(GameComponent)]
#[repr(u8)]  // Always 1 byte
pub enum GoodEnum {
    A = 0,
    B = 1,
}
```

---

### 3. String Handling

```rust
// BAD: String in FFI struct
#[derive(GameComponent)]
pub struct BadString {
    pub name: String,  // Won't work!
}

// GOOD: Fixed-size buffer or UUID
#[derive(GameComponent)]
pub struct GoodString {
    pub name: [u8; 32],  // Fixed-size UTF-8
    pub name_len: u32,
}
```

```rust
    #[field(min = 0.0, max = 100.0)]
    pub maximum: f32,
}

#[derive(GameComponent)]
#[uuid = "00000006-0000-0000-0000-000000000006"]
#[reflect]
pub struct PlayerState {
    pub is_alive: bool,
    #[field(skip)]
    pub last_damage_time: f32,
    #[field(skip)]
    pub damage_history: Vec<(Uuid, f32)>,
}

// === Systems ===

#[derive(GameSystem)]
#[system(stage = Update)]
fn movement_system(
    mut query: Query<(&mut Position, &Velocity)>,
    time: Res<Time>,
) {
    for (mut pos, vel) in query.iter_mut() {
        pos.x += vel.x * time.delta_seconds();
        pos.y += vel.y * time.delta_seconds();
        pos.z += vel.z * time.delta_seconds();
    }
}

#[derive(GameSystem)]
#[system(stage = Update)]
fn health_regen_system(
    mut query: Query<&mut Health>,
    time: Res<Time>,
) {
    for mut health in query.iter_mut() {
        if health.current < health.maximum {
            health.current += 5.0 * time.delta_seconds();
            health.current = health.current.min(health.maximum);
        }
    }
}

#[derive(GameSystem)]
#[system(stage = Update)]
fn sync_to_engine(
    query: Query<(Entity, &Position), Changed<Position>>,
    mut writer: FfiWriter,
) {
    for (entity, pos) in query.iter() {
        writer.write_component(entity.id(), pos);
    }
}

// === Registration ===

fn setup_game(world: &mut World) {
    let mut registry = ReflectRegistry::new();
    
    // Register all components
    registry.register::<NetworkId>();
    registry.register::<Position>();
    registry.register::<Rotation>();
    registry.register::<Velocity>();
    registry.register::<Health>();
    registry.register::<PlayerState>();
    
    world.insert_resource(registry);
}

// === Spawning ===

fn spawn_player(commands: &mut Commands, id: Uuid) -> Entity {
    commands.spawn().insert_bundle((
        NetworkId { id },
        Position { x: 0.0, y: 0.0, z: 0.0 },
        Rotation { x: 0.0, y: 0.0, z: 0.0, w: 1.0 },
        Velocity { x: 0.0, y: 0.0, z: 0.0 },
        Health { current: 100.0, maximum: 100.0 },
        PlayerState { 
            is_alive: true,
            last_damage_time: 0.0,
            damage_history: Vec::new(),
        },
    )).id()
}
```
